# Studio_Namma_Clone

An interactive landing page inspired by Studio Namma, built using HTML, CSS, and JavaScript.

## Features

- Custom mouse-following cursor
- Cursor-following video preview in the hero section
- Animated fullscreen overlay menu
- Interactive project showcase with hover video previews
- Dynamic image stack animations on text hover
- Service section with hover-based video reveals
- Responsive layout structure
- Smooth hover transitions and micro-interactions

## Built With

- HTML5
- CSS3
- JavaScript

## Project Structure

```text
├── index.html
├── style.css
├── script.js
└── assets
```

## What I Practiced

- DOM Manipulation
- Event Handling
- Mouse Tracking Effects
- Hover Animations
- CSS Transforms & Transitions
- Interactive UI Development

## Run Locally

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the project folder

```bash
cd studio_namma_clone
```

3. Open `index.html` in your browser or use Live Server.
